# Сайт о певце SHAMAN

Небольшой статичный сайт на **HTML** и **CSS**, посвящённый российскому певцу SHAMAN.  
Страница содержит несколько фотографий артиста, под каждой из которых размещено краткое текстовое описание: кто он такой, основные факты о карьере и творчестве, интересные детали биографии.  

На сайте также воспроизводится фоновая музыка.  
Для удобства пользователя реализован **ползунок громкости**, с помощью которого можно регулировать уровень звука, не покидая страницу.

## Структура проекта

- `index.html` — основная страница сайта  
- `style.css` — стили для оформления разметки  
- `images/` — папка с фотографиями SHAMAN  
- `audio/` или `music/` — папка с аудиофайлом фоновой музыки (имя и путь должны совпадать с указанными в `index.html`)  

## Запуск

1. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge и т.д.).  
2. Убедитесь, что все изображения лежат в папке `images/` и пути к ним в коде указаны верно.  
3. Проверьте, что аудиофайл фоновой музыки находится в соответствующей папке и корректно подключён в `index.html`.  

## Назначение

Проект создан в учебных целях для практики вёрстки на HTML и CSS, а также работы с мультимедийными элементами (изображения и аудио) в вебе.
